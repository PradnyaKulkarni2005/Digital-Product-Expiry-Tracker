# Digital Product Expiry Tracker

A full-stack web application that helps users track product expiry dates by automatically extracting expiry information from uploaded product images. The system sends timely alerts when products are nearing expiration, helping users reduce waste and manage household or inventory items efficiently.

---

## 🚀 Features

- 📸 Upload product images containing expiry date labels
- 🔍 Extract expiry dates automatically using OCR and Regular Expressions
- 📅 Store and manage products in a centralized dashboard
- 🔔 Receive alerts when products are close to expiration
- 👤 Secure user authentication and personalized product lists
- 🗂️ Search, update, and delete tracked products
- 📊 View active, expiring, and expired products

## 🏗️ System Workflow

1. User registers and logs into the application.
2. User uploads an image of a product label.
3. OCR extracts text from the image.
4. Regex identifies the expiry date from the extracted text.
5. Product details and expiry dates are stored in MySQL.
6. The dashboard displays all tracked products.
7. The system generates alerts for products nearing expiration.

---

## 💡 Real-World Impact

Managing expiry dates manually can be difficult, especially for groceries, medicines, and cosmetics. This application automates the process, helping users:

- Reduce product waste
- Avoid using expired products
- Improve inventory organization
- Save time and effort
